English(Translated)
Novagia NVG Mini Runtime was created to simplify organization and integration between different programming languages (inspired by the Love2D system).
How to use:
Create a main.cs file that will call your other files (or just use the main.cs itself).
For example, if you have a main.cs that calls two Python scripts: compress the main.cs into a .zip file along with the two Python scripts, DLL folders, libraries, and everything else.
Next, rename the .zip file extension to .nvg.
Open D.T.I.exe.
Specify the path to your .nvg file and click Run.
It will manage everything for you, and your application will open with all scripts running. It’s like an .exe, but easier to use. Instead of having messy files and a loose .exe in a folder, just compress everything, turn it into an .nvg, and run it using the desnan.dll (check the source code).
Note: This was made by a 10-year-old as a study on DLLs, so don't expect it to be a professional-grade tool.
DON'T FORGET!
  The core project is the desnan.dll; the D.T.I.exe is just the interface to interact with it.
